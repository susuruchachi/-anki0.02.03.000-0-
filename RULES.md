## 🗂️ 画面構成

すするankiは下部ナビゲーションバーから各画面に移動できます。

| ナビ | 画面 | 主な用途 |
|------|------|---------|
| 🏠 ホーム | ホーム画面 | クイズ開始・スコープ設定 |
| 📁 フォルダー | 構造エクスプローラー | フォルダー管理 |
| 📝 カード箱 | 問題カード箱 | カード一覧・検索 |
| ⚔️ オンライン対戦 | オンライン対戦 | リアルタイム対戦 |
| 📊 実績 | 習熟実績と学習状況 | 学習グラフ・統計 |
| 💾 管理・設定 | 管理・クラウド同期 | 設定・データ管理 |

## 📝 カード（問題と答え）

すするankiの最小単位。「問題文」と「答え」で構成されます。

### カード操作

- **短いタップ** → 答えの表示/非表示を切り替え
- **長押し（500ms）** → 編集・削除・移動メニューを表示
  - ✏️ 問題・答えの編集
  - 📁 別フォルダーへ移動
  - ❌ 削除（元に戻せません）

### カード追加

- **「➕ 新規追加」ボタン** → 1枚ずつ追加（フォルダー指定可）
- **一括追加** → フォルダーを長押し →「➕ 問題を一括追加 (Q,A 改行)」
  - 書式：奇数行が問題、偶数行が答え（交互に入力）
- **📥 インポート** → 管理・設定画面からJSONまたはCSVで一括インポート

### 答えの複数表記・正規化

`/` または `|` で区切ると複数の答えをどれでも正解と判定できます。

例：`東京/Tokyo` → 「東京」でも「tokyo」でも正解

ひらがな・カタカナ・全角/半角・大文字/小文字は同一視されます。

### クイックメモ

クイズ中に答えを見た後、語呂合わせ等をメモ欄に記録できます。次回出題時に表示されます。

## 📂 フォルダー（カテゴリー）

カードを整理するための階層構造。「未分類」は特別なフォルダーで変更・削除できません。

### フォルダー作成

- **構造エクスプローラー上部の入力欄** → ルートフォルダーを新規作成
- **長押しメニュー→「➕ 中にサブフォルダーを作る」** → 入れ子フォルダーを作成

### フォルダーの長押しメニュー

フォルダーを長押しすると以下のメニューが表示されます：

| 項目 | 説明 |
|------|------|
| 🔗 共有URLを発行 | フォルダーを友人と共有するURLを生成 |
| 🌐 公開カテゴリーに設定 | 全ユーザーが購読できる公開リストに登録 |
| ➕ 中にサブフォルダーを作る | 入れ子フォルダーを作成 |
| 📝 この直下に問題を追加 | そのフォルダーにカードを1枚追加 |
| ✏️ フォルダー名を変更 | リネーム（中のカードも自動更新） |
| ➕ 問題を一括追加 | Q/A交互の書式で複数カードを一度に追加 |
| 📁 「○○」の中へ移動 | 別フォルダーの配下に移動 |
| 🔄 成績をリセット | そのフォルダー内の学習記録だけをリセット |
| ❌ 削除 | フォルダーと中身をすべて削除 |
| 🚫 購読を解除 | 他人が作った購読フォルダーを解除して削除 |

> **購読カテゴリーの場合**、「❌ 削除」の代わりに「🚫 購読を解除」が表示されます。

## ❓ クイズモード

ホーム画面から出題範囲を選んでクイズを開始します。

### 出題範囲の選択

ホーム画面上部のドロップダウンで出題するフォルダーを絞り込めます（階層をドリルダウンして選択可能）。

### 出題形式（クイズモード）

| モード | 説明 |
|--------|------|
| 📝 記述式入力 | キーボードで答えを直接入力（全角/半角不問） |
| 🎴 4択択一 | 4つの選択肢から正解を選ぶ |
| ⚡ みんはや形式 | 答えを1文字ずつ4択で選んでいく（区切り文字は自動スキップ） |
| 👆 文字タップ | バラバラになった文字をタップして並べる（Duolingo風） |
| 🧠 自己申告 | 答えを見て自分で「❌ ミス / △ 普通 / ⭕ 完璧」を申告（Anki風） |

### 起動ボタンの種類

| ボタン | 説明 |
|--------|------|
| ▶️ クイズ開始 | 通常のSRSクイズ（弱いカードを優先出題） |
| 🔥 集中特訓 | 苦手・しっかりマークのカードのみ出題 |
| ♻️ ランダム復習 | 卒業済みカードからランダム出題 |

### クイズオプション

- **🔄 問題と答えを入れ替えて出題**（逆引きモード）：答えが問題文になり、問題文を答えとして入力する
- **🎓 卒業済みのカードも出題に含める**：チェックを入れると卒業カードも出題対象に含む
- **出題数**：1回のクイズで出題する問題数を指定
- **卒業認定正解数**：何回正解でカードが卒業するか（デフォルト5回）
- **時間制限（秒）**：0にすると無制限
- **自動進行（秒）**：答え表示後、自動で次の問題へ進む秒数（0で手動）

### クイズ中の操作

- **🏳️ パス** → その問題をスキップ（不正解扱いにならない）
- **確定する** → 回答を送信
- **クイックメモ欄** → 語呂合わせ等を記録（次回出題時も表示）

## 🎯 SRS（間隔反復）システム

正解・不正解に応じてカードのレベルが変動し、弱いカードが優先的に出題されます。

### カードのレベルとステータス

| ステータス | 条件 | 出題優先度 |
|-----------|------|-----------|
| 🎓 卒業 (超安定) | 正解数が卒業基準以上 | 低（オプションで含める） |
| ⚡ 得意 (Lv3〜5) | レベル3以上 | 普通 |
| ☕ 普通 (Lv1〜2) | レベル1〜2 | 普通 |
| ⚠️ 苦手 (Lv0) | レベル0で実績あり | 高 |
| 🧱 しっかり (Lv-1) | 連続不正解で降格 | 最高 |
| 💤 未着手 | まだ1度も出題されていない | 中 |

### レベルの変動ルール

- **正解が2連続** → レベルアップ（最大Lv5）
- **不正解が2回ごと** → レベルダウン
- **不正解が4連続** → しっかり（Lv-1）に降格・正解数リセット
- **しっかり状態で5連続正解** → Lv0に復帰

> 設定の「卒業認定正解数」を変えると、上記の閾値もそれに比例してスケールします。

## 🌐 公開カテゴリー

全ユーザーが閲覧・購読できるカテゴリーリストです。

### 公開する

1. フォルダーを長押し →「🌐 公開カテゴリーに設定」
2. 「カード箱」画面上部の「🌐 公開カテゴリーを見る」から全ユーザーに表示される

### 購読する

1. 「カード箱」画面→「🌐 公開カテゴリーを見る」
2. カテゴリーを選んで「📥 購読してインポート」
3. 以後、作成者の更新が自動同期される

### 購読を解除する

1. 構造エクスプローラーで対象フォルダーを長押し
2. 「🚫 購読を解除」をタップ
3. フォルダーと中の問題がすべて削除され、自動同期も停止します

### 公開を停止する（作成者のみ）

「🌐 公開カテゴリーを見る」ページで自分のカテゴリーの「🗑️ 削除」ボタンをタップ。
手元のデータは残ります。

## 🔗 カテゴリー共有（URLシェア）

フォルダーを特定の人と共有します（公開リストには載りません）。

### 共有する

1. フォルダーを長押し →「🔗 このフォルダーの共有URLを発行」
2. 生成されたURLまたは6文字コードを相手に送信
3. 相手がURLにアクセスして購読 → 自動で問題が同期される

### 共有の権限

| 権限 | できること |
|------|----------|
| 作成者 | カードの追加・編集・削除、共同編集者の設定 |
| 共同編集者 | カードの追加・編集・削除 |
| 購読のみ | 学習のみ（問題の編集不可） |

### 共同編集者の設定

「管理・設定」画面→「共有カテゴリー管理」→「👑 共同編集者の設定」で相手のUIDを追加します。

## ⚔️ オンライン対戦

リアルタイムで他のユーザーと問題を出し合うモードです。

### 対戦の流れ

1. 「⚔️ オンライン対戦」ページを開く
2. 出題するフォルダーとクイズ形式を選択
3. 「対戦部屋を作成」→ 相手に部屋IDを共有
4. 相手が「部屋に参加」→ 両者同時に問題が出題される
5. 先に多く正解した方が勝ち

### 途中離脱・復帰

対戦中に別のページに移動しても、対戦は継続されます。オンライン対戦ページに戻ると自動的に対戦画面に復帰します。

## 👥 フレンド・ランキング機能

### フレンド登録

1. 「管理・設定」→「🏆 ランキング・チャットを開く」→「フレンド追加」タブ
2. 相手のUID（管理タブでコピー可能）を入力
3. 相互登録が完了するとフレンド一覧に表示

### ランキング

- **デイリーランキング** → 今日の正解数トップ10
- **カテゴリー別ランキング** → フレンドとの習熟度比較

### チャット

フレンドとのダイレクトチャットが可能です。

### 成績共有の ON/OFF

「管理・設定」画面の「成績共有」チェックボックスで OFF にするとランキングに表示されなくなります。

## 💾 データ管理

### ☁️ クラウド同期（Google）

Googleアカウントでログインすると、複数デバイス間でデータを自動同期します。

- **手動同期**：「☁️ クラウドと手動マージ同期」ボタン
- 自動同期はカード追加・編集・削除のたびに行われます

### 📥 インポート

| 形式 | 内容 |
|------|------|
| JSON | カード・フォルダー構造・学習実績をすべて復元 |
| CSV | カードのみ（書式：問題,答え,フォルダー名） |

### 📤 エクスポート

| 形式 | 内容 |
|------|------|
| JSON | 全データの完全バックアップ |
| CSV | スプレッドシートで開ける形式 |

### ⚙️ リセット

- **成績リセット** → カード・フォルダーを保持し、正解数・レベルのみリセット
- **全消去** → すべてのデータを削除（復元不可）

## ⚠️ 注意事項

### データの扱い

- **ローカルカテゴリー** → 自由に編集・削除可能
- **購読カテゴリー（閲覧専用）** → カードの編集・削除不可（フォルダー構造の変更も不可）
- **購読カテゴリー（共同編集者）** → カードの追加・編集・削除が可能
- **削除したカード** は元に戻せません。定期的にJSON形式でバックアップを推奨します

### クラウド同期について

- 同期はマージ方式（ローカルとクラウドの和集合を保持）
- 同一IDのカードはクラウド側を正とする
- 共有カテゴリーのカードは作成者の変更が自動反映される

## 🆘 トラブルシューティング

| 症状 | 対処法 |
|------|-------|
| ボタンが反応しない | ブラウザキャッシュをクリア → リロード |
| クラウド同期されない | ネット接続・Googleログイン状態を確認 |
| 共有カテゴリーが編集できない | 閲覧専用が正常動作。共同編集者登録を依頼してください |
| 購読を解除できない | フォルダーを長押し →「🚫 購読を解除」を選択 |
| 対戦相手が表示されない | 両者が同じ部屋IDで接続しているか確認 |
